# ========================================
# Notes & Best Practices
# ========================================

# 1. SECURITY:
#    - NEVER commit .env file to git
#    - NEVER share private keys
#    - Use hardware wallets for mainnet
#    - Rotate keys regularly

# 2. PAUSER ADDRESSES:
#    - Use different addresses from deployer
#    - Distribute pausers to trusted parties
#    - Keep private keys secure
#    - Test pause functionality before mainnet

# 3. RPC PROVIDERS:
#    - Infura: https://infura.io (free tier available)
#    - Alchemy: https://alchemy.com (free tier available)
#    - QuickNode: https://quicknode.com (paid)
#    - Public RPCs: Often rate-limited

# 4. GAS OPTIMIZATION:
#    - Higher OPTIMIZER_RUNS = better for frequently called functions
#    - Lower OPTIMIZER_RUNS = cheaper deployment
#    - Recommended: 200-1000 for most contracts

# 5. TESTING:
#    - Get Sepolia ETH from faucets:
#      - https://sepoliafaucet.com
#      - https://faucet.quicknode.com/ethereum/sepolia
#    - Test thoroughly on testnet before mainnet

# ========================================
# Setup Instructions
# ========================================

# Step 1: Copy this file
#   cp .env.example .env

# Step 2: Get Sepolia RPC URL
#   - Sign up at Infura.io or Alchemy.com
#   - Create new project
#   - Copy Sepolia endpoint URL

# Step 3: Export Private Key
#   - Open MetaMask
#   - Account Details > Export Private Key
#   - Paste in PRIVATE_KEY field (include 0x prefix)

# Step 4: Configure Pausers
#   - Generate 3+ addresses (can use MetaMask)
#   - Add addresses to PAUSER_ADDRESS_0, 1, 2...
#   - Update NUM_PAUSERS to match

# Step 5: Get Sepolia ETH
#   - Visit https://sepoliafaucet.com
#   - Request test ETH for deployer address

# Step 6: Deploy Contract
#   npm run deploy

# Step 7: Update CONTRACT_ADDRESS
#   - Copy address from deployment output
#   - Paste in CONTRACT_ADDRESS field

# Step 8: Verify Contract (Optional)
#   - Get Etherscan API key
#   - Add to ETHERSCAN_API_KEY
#   - Run: npx hardhat verify --network sepolia <address>
